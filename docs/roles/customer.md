# Customer Guide

Use the customer account to browse products, manage saved checkout addresses, request scheduled consultant appointments, and review orders. Checkout keeps a snapshot of the selected address on each order.

Customer data is private to the signed-in account. Saved addresses are optional; guests can still enter a one-time shipping address.
