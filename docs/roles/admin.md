# Admin Guide

Use the admin panel to manage catalog content, media, astrologer accounts, orders, payments, support, integrations, audit logs, and project health. Configure payment mode and credentials in Admin -> Integrations before enabling customer payments.

This guide covers operating tasks only. Internal architecture and deployment details belong in the engineering documentation.
