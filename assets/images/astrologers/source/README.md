# Client Astrologer Source Images

- These 21 JPEG files are byte-for-byte originals from `astrologer's card-20260619T101642Z-3-001.zip` supplied by the client.
- Stable filenames match the astrologer slugs in `storage/data/astrologers.json`.
- Public portraits in `../client/` are generated from crop box `(305, 200, 775, 670)` on each 1080x1080 original, then resized to 420x420. This slight inset removes the external red card background from the circular frame.
- Do not crop the upper-left 420px square; it contains template background and cuts off the portrait.
